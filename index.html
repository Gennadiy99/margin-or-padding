5.5 margin или padding, Что лучше использовать

В этом уроке я расскажу, где лучше использовать margin, а где padding. В каких ситуациях и как лучше использовать эти два свойства margin и padding для создания отступов. Организация отступов margin и padding очень важна и на первых этапах может возникать вопрос, что же лучше использовать margin или padding в той или иной ситуации. В этом уроке я расскажу о нескольких простых правилах, которые помогут вам в организации отступов с помощью margin и padding.

Для разделения элементов Друг от Друга нужно использовать "margin". Отступы всегда идут от предыдущего к следующему. Так работают отступы для элементов

В текстовых тегах имеются по умолчанию "margin" поэтому часто в стилях CSS пишут сброс для свех тегов: 

        * {
          margin: 0;
          padding: 0;
        }

Для того что бы отодвинуть Текст сверху и с лева используют "padding" 

        h1 {
          padding-top: 15px;
          padding-left: 15px;
        }

"margin" - Нужно задавать Соседним элементам, а не дочерним.

Если планируется использовать класс: "list" в другом месте кода но без отступов "margin" то нужно создать другой класс " mr5" в котором описать отступы, и не прописывать их в класс: "list"

Еще можно создать див обертку с классом ".mr5" И тогда не нужно указывать 2 класса "list mr5" Обертка будет устанавливать отступы.

Отступы для последнего элемента в группе нужно обнулять! 

        .menu li:last-child{
          margin-right: 0;
        }

Отступ между элементами "li" и картинкой нужно задавать не через дочерние а через родительские элементы. То есть "margin" указывать для блока "menu" в котором находятся "li"

Есть интереная фишка записи отступов, где отступы "margin" указываются соседним элементам ".mr5": 

        .mr5 + .mr5{
          margin-top: 15px;
        }

Выше указанная запись заменяет запись где каждому ".mr5" устанавливали верхний отступ "margin-bottom: 15px"; а топотом последнему ".mr5:last-child" элементу отменяли отступ "margin-bottom: 0;": 

        .mr5 {
          margin-bottom: 15px;
        } 
        .mr5:last-child{
          margin-bottom: 0;
        }
